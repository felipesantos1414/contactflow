# 🚀 ContactFlow

Smart Contact Management System built with HTML, CSS and JavaScript.

---

## 📌 Overview

ContactFlow is a lightweight web application designed to organize and manage contacts efficiently.

It allows users to quickly search, filter and interact with contacts through a clean and responsive interface.

This project was built as part of my transition into the tech field, focusing on practical solutions and real-world usability.

---

## 🔧 Features

- 🔍 Real-time search  
- 🎯 Filter by contact type  
- 🔤 Sort contacts by name  
- 📋 Copy contact information with one click  
- 📱 Direct WhatsApp access  
- ⚡ Fast and lightweight (no frameworks)  
- 💻 Responsive design  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- JSON (data structure)
